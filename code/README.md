# AlinD OS

![AlinD OS Desktop Mockup](file:///C:/Users/emina/.gemini/antigravity/brain/131e1543-01fe-4c38-a4bb-d309e984d1c6/alind_os_desktop_mockup_1769795143500.png)

**AlinD OS**, Linux çekirdeği temel alınarak geliştirilen, modern, minimalist ve kullanıcı dostu bir işletim sistemi deneyimidir. 

"0 Uygulama" felsefesiyle tasarlanan AlinD OS, gereksiz karmaşıklıktan arındırılmış, sadece ihtiyacınız olan araçları (Terminal, Dosya Yöneticisi, Ayarlar) sunan ve hıza odaklanan bir sistemdir.

## Web Presence

- **[Official Landing Page](landing/index.html)**: Get to know AlinD OS and download the latest ISO.

- **Minimalist ADE (AlinD Desktop Environment):** Cam efekti (Glassmorphism) ve modern görsel dil.
- **Mac-Style Launcher:** `Super (Win) + Space` tuş kombinasyonu ile açılan merkezi uygulama başlatıcı.
- **Linux Core:** Güçlü, stabil ve güvenli Linux çekirdeği.
- **Developer Friendly:** Sadece gerekli araçlar, sıfır bloatware.

## Getting Started

AlinD OS'u denemek veya geliştirmeye katkıda bulunmak için:

1. Depoyu klonlayın: `git clone https://github.com/kullaniciadi/AlinD-OS.git`
2. `desktop/index.html` dosyasını tarayıcınızda açarak arayüzü inceleyin.
3. Linux ISO entegrasyonu için `docs/ISO_GUIDE.md` dosyasını takip edin.

## Shortcuts

- `Super + Space`: AlinD Launcher
- `Super + T`: Terminal (Yakında)
- `Super + E`: Dosya Yöneticisi (Yakında)

## Project Structure

- `/desktop`: AlinD Desktop Environment (Vanilla JS/CSS)
- `/experimental`: Eski Win32 ve 16-bit ASM denemeleri (Legacy)
- `/docs`: Kurulum ve geliştirme dokümanları
